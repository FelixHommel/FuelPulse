# ADR 002: Use an Event Bus

## Status

*Accepted*

## Context

Components such as the ```scheduler```, ```collector```, ```logger```, and ```analyzer``` should not depend directly on
each other.

## Decision

Components communicate through published events. Such events could include:

- ```MinuteTick```
- ```CollectionFinished```
- ```DailyAnalysisRequested```
- ```StartRequested```
- ```ShutdownRequested```

## Consequences

### Pros

- Loose coupling
- Easier testing
- Components can be added without modifying existing code

### Cons

- Harder to trace event flow during debugging
