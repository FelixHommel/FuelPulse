# ADR 003: Store Raw Data in SQLite

## Status

*Accepted*

## Context

Collected measurements must survive restarts and allow historical reanalysis.

## Decision

Persist all raw measurements in SQLite. Analysis reads from SQLite rather than directly from the collector.

## Consequences

### Pros

- Can recompute analyses
- Small deployment foorprint
- Good performance for the workload

### Cons

- One additional dependency
